# Snack is a RPC Framework
* Snack 基于Java语言编写，是基于Spring-boot实现的微服务RPC框架。
* 网络通信基于Netty4。
* 消息序列化使用ProtoStuff。
* 服务治理基于Zookeeper。
* 支持轮询调度、随机调度。
